FmToast 轻提示组件 ​

全局轻提示组件，用于展示操作反馈消息。

基础用法 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function showToast() {
  toast.success('操作成功！')
}
</script>

<template>
  <FmButton @click="showToast">显示成功提示</FmButton>
  <!-- 在应用根组件中添加 -->
  <FmToast />
</template>

Props ​

无（通过 toast 函数调用配置）

Slots ​

无

toast 方法 ​

方法说明
toast(message)普通提示
toast.success(message)成功提示
toast.error(message)错误提示
toast.warning(message)警告提示
toast.info(message)信息提示
toast.loading(message)加载提示
toast.promise(promise, options)Promise 提示

示例 ​

成功提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleSuccess() {
  toast.success('保存成功！')
}
</script>

<template>
  <FmButton @click="handleSuccess">保存</FmButton>
</template>

错误提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleError() {
  toast.error('操作失败，请稍后重试')
}
</script>

<template>
  <FmButton @click="handleError">触发错误</FmButton>
</template>

警告提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleWarning() {
  toast.warning('数据即将过期，请及时更新')
}
</script>

<template>
  <FmButton @click="handleWarning">警告提示</FmButton>
</template>

信息提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleInfo() {
  toast.info('系统将于今晚 22:00 进行维护')
}
</script>

<template>
  <FmButton @click="handleInfo">信息提示</FmButton>
</template>

加载提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleLoading() {
  const loadingToast = toast.loading('正在处理...')
  
  setTimeout(() => {
    toast.dismiss(loadingToast)
    toast.success('处理完成！')
  }, 2000)
}
</script>

<template>
  <FmButton @click="handleLoading">加载提示</FmButton>
</template>

Promise 提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

async function handlePromise() {
  toast.promise(
    new Promise((resolve) => {
      setTimeout(resolve, 2000)
    }),
    {
      loading: '正在保存...',
      success: '保存成功！',
      error: '保存失败',
    }
  )
}
</script>

<template>
  <FmButton @click="handlePromise">Promise 提示</FmButton>
</template>

带描述的提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleDescription() {
  toast.success('上传成功', {
    description: '文件已上传到服务器',
  })
}
</script>

<template>
  <FmButton @click="handleDescription">带描述的提示</FmButton>
</template>

带操作的提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'
import { h } from 'vue'

function handleAction() {
  toast.success('删除成功', {
    action: {
      label: '撤销',
      onClick: () => {
        // 撤销操作
        console.log('撤销删除')
      },
    },
  })
}
</script>

<template>
  <FmButton @click="handleAction">带操作的提示</FmButton>
</template>

多个提示 ​

vue
<script setup lang="ts">
import { toast } from 'vue-sonner'

function handleMultiple() {
  toast.success('第一条消息')
  setTimeout(() => {
    toast.error('第二条消息')
  }, 500)
  setTimeout(() => {
    toast.warning('第三条消息')
  }, 1000)
}
</script>

<template>
  <FmButton @click="handleMultiple">显示多个提示</FmButton>
</template>

全局配置 ​

在应用根组件中配置全局选项：

vue
<script setup lang="ts">
import { Toaster } from 'vue-sonner'
</script>

<template>
  <Toaster 
    position="top-center"
    :duration="3000"
    :visible-toasts="3"
  />
</template>

注意事项 ​

  1. 全局组件：<FmToast /> 需要在应用根组件中添加一次
  2. toast 函数：通过 import { toast } from 'vue-sonner' 导入使用
  3. 自动消失：提示默认 3 秒后自动消失
  4. 堆叠显示：多个提示会堆叠显示
  5. 可点击关闭：每个提示都可以手动点击关闭

典型使用场景 ​

  • 表单提交反馈
  • 操作成功/失败提示
  • 删除确认反馈
  • 上传/下载进度提示
  • 系统通知
  • 数据保存提示
